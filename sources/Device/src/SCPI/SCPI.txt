Команды должны заканчиваться 0x0D, 0x0A в любых комбинациях.
Строчные и прописные символы не различаются.

*IDN?                                           // Запрос идентификатора
*RST                                            // Сброс на начальные установки
RANGE {2mA | 20mA | 200mA | 2A | 20A | 50A}     // Выбор предела
DATA?                                           // Запросить одно измерение
DATA? [1...(int)0xFFFFFFFF]                     // Запросить N измерений
AVP  {ON | OFF}                                 // Вкл/откл АВП
